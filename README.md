## 硬件功能

1. Arduino通过I2C读取MPU9250模块9轴数据；
2. 对数据添加校验位并进行编码，之后通过串口发送；
3. 采用一块7.4V锂电池通过DC接口(7~12V)进行供电；

## 上位机功能

1. 接收硬件数据，解码后得到正确的传感器数据；
2. 进行AHRS解算，得到姿态角；
3. 姿态角的实时三维显示；
